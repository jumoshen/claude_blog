# 博客系统产品文档

## 1. 产品概述

一个简洁的个人博客系统，包含面向读者的文章展示端和面向运营者的管理后台。

### 1.1 产品定位

- **C端（展示端）**：服务于文章读者，提供流畅的阅读体验
- **B端（管理后台）**：服务 于运营者，提供便捷的内容管理能力

### 1.2 用户群体

- 博主/内容创作者：撰写、发布、管理文章
- 技术读者：浏览、学习、阅读技术文章

---

## 2. C端功能

### 2.1 文章列表页

读者进入博客后看到的首个页面，集中展示所有已发布的文章。

**功能要点：**
- 文章列表按发布时间倒序排列
- 每篇文章展示：标题、发布日期、标签、浏览数
- 点击标题可进入文章详情页

### 2.2 文章详情页

文章的核心阅读页面，承载全部内容。

**功能要点：**
- Markdown 内容渲染，代码块支持语法高亮
- 文章头部显示：发布日期、标签、浏览数
- 每次访问自动记录一条访问日志

### 2.3 访问统计

帮助博主了解文章的阅读情况。

**功能要点：**
- 记录每次访问的：IP、User-Agent、访问时间
- 支持按时间范围筛选访问记录

---

## 3. B端功能

### 3.1 认证模块

管理后台的入口控制。

**功能要点：**
- 管理员账号登录/登出
- 基于 JWT 的身份认证

### 3.2 文章管理

后台的核心功能模块。

**功能要点：**

| 功能 | 说明 |
|-----|------|
| 文章列表 | 分页展示，支持按状态筛选（草稿/已发布/已下架） |
| 创建文章 | Markdown 编辑器，支持代码高亮预览 |
| 编辑文章 | 延续创建流程，修改后保存 |
| 发布/下架 | 一键切换文章状态 |
| 删除文章 | 软删除，保留数据仅标记删除 |
| Markdown 导入 | 批量导入本地 Markdown 文件 |
| 模版下载 | 下载标准 Markdown 写作模版 |
| 数据初始化 | 从历史数据批量导入已有文章 |

### 3.3 用户管理

管理博客的注册用户。

**功能要点：**
- 查看用户列表：展示用户 ID、昵称、注册时间等信息
- 查看用户详情：查看用户完整信息
- 编辑用户信息：修改昵称、GitHub 关联信息等

### 3.4 管理员管理

管理后台自身的管理员账号。

**功能要点：**
- 管理员列表：增删改查
- 密码修改：修改自身密码

### 3.5 访问统计

帮助运营者了解博客的流量情况。

**功能要点：**
- **数据概览**：展示总访问量、今日访问量、热门文章 TOP5
- **访问记录**：按时间查询详细访问日志
- **文章排行**：按浏览数降序排列的文章列表
- **可视化图表**：使用 ECharts 展示统计数据

### 3.6 文件管理

文章图片上传服务。

**功能要点：**
- 支持格式：jpg、png、gif、webp、svg
- 单文件限制：最大 10MB
- 访问权限：公开访问
