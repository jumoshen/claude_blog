# Lucky Draw Tool Design - 幸运抽奖工具

## 1. 概述

在C端文章页面新增一个可收起的工具箱，包含骰子和翻牌两个独立工具。

## 2. 交互设计

### 工具箱组件 (Toolbox)

- **位置**：左下角固定悬浮
- **收起状态**：显示一个小图标按钮（扑克牌/骰子图标）
- **展开触发**：hover展开
- **展开动画**：smooth transition，工具从按钮中滑出
- **展开内容**：
  - 骰子工具（保留原有功能）
  - 翻牌工具（扑克牌图标）

### 翻牌组件 (LuckyCard)

#### 触发方式
- 点击工具箱中的翻牌工具按钮

#### 弹出行为
- 从下方滑入动画，弹出3张牌
- 牌背面朝上，显示扑克牌图案

#### 牌背面设计（跟随主题）

**像素风**
- 深色背景 + 像素化扑克牌图案
- 棋盘格纹理边框
- 复古游戏机风格

**可爱风**
- 马卡龙色系渐变背景
- 圆润的扑克牌轮廓
- 粉色/浅紫色调

**Q版**
- 赛博朋克风霓虹边框
- 深色背景 + 发光效果
- 科技感几何图案

#### 翻牌动画
- 3D翻转效果 (rotateY)
- 翻转时长 0.8s
- 弹性缓动函数
- 依次翻牌，每张间隔 0.2s

#### 翻牌正面
- 显示文章标题
- 标题过长时截断+省略号
- 点击标题跳转详情页

## 3. 技术实现

### 组件结构

```
Toolbox.vue (工具箱容器)
├── DiceTool (骰子 - 复用现有组件)
└── CardTool (翻牌工具按钮)
    └── LuckyDrawModal (翻牌弹窗)
        └── Card (单张扑克牌 × 3)
```

### 关键文件

- `blog-frontend/src/components/common/Toolbox.vue` - 新增工具箱容器
- `blog-frontend/src/components/common/LuckyDrawModal.vue` - 新增翻牌弹窗
- `blog-frontend/src/components/common/ReadingProgress.vue` - 修改，移除独立骰子按钮

### 动画实现

- 3D翻牌使用 CSS `transform: rotateY()` + `perspective`
- 使用 `backface-visibility: hidden` 实现正反面
- 工具箱展开使用 CSS transition

## 4. 主题适配

| 主题 | 牌的边框 | 背景纹理 | 主色调 |
|------|---------|---------|-------|
| 像素风 | 棋盘格 | 像素点阵 | 紫色系 |
| 可爱风 | 圆润边框 | 渐变圆点 | 粉色系 |
| Q版 | 霓虹发光边框 | 几何线条 | 蓝紫系 |
