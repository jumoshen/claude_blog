# T17: 文章密码保护

## 需求分析
允许博主为文章设置密码保护，访问时需要输入密码。

## 实现方案

### 1. 数据库设计
```sql
ALTER TABLE posts ADD COLUMN password_hash VARCHAR(255) DEFAULT '' COMMENT '文章密码保护';
```

### 2. API 设计
| 方法 | 路径 | 说明 |
|-----|------|------|
| POST | /api/v1/posts/:slug/verify | 验证文章密码 |

### 3. 业务逻辑
- 有密码的文章返回 403 + 需要验证
- 密码验证通过后设置 cookie/session
- 前端需处理密码验证流程
