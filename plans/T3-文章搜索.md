# T3: 文章搜索（标题、内容关键字）

## 需求分析
提供文章搜索功能，支持按标题和内容关键字搜索。

## 实现方案

### 1. API 设计

| 方法 | 路径 | 说明 |
|-----|------|------|
| GET | /api/v1/posts/search | 搜索文章 |

**查询参数：**
- `q`: 搜索关键字（必填）
- `page`: 页码，默认1
- `page_size`: 每页数量，默认10

### 2. 业务逻辑
- 使用 MySQL LIKE 查询
- 搜索 title 和 content 字段
- 按相关度/发布时间排序
- 支持分页
