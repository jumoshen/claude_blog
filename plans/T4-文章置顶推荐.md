# T4: 文章置顶/推荐

## 需求分析
支持文章置顶和推荐标记，让重要文章更容易被看到。

## 实现方案

### 1. 数据库设计
```sql
ALTER TABLE posts ADD COLUMN is_pinned TINYINT(1) DEFAULT 0 COMMENT '是否置顶';
ALTER TABLE posts ADD COLUMN is_featured TINYINT(1) DEFAULT 0 COMMENT '是否推荐';
ALTER TABLE posts ADD COLUMN pinned_at DATETIME DEFAULT NULL COMMENT '置顶时间';
```

### 2. API 设计

| 方法 | 路径 | 说明 |
|-----|------|------|
| GET | /api/v1/posts/featured | 获取推荐文章列表 |
| PUT | /api/admin/posts/:id/pin | 设置/取消置顶 |
| PUT | /api/admin/posts/:id/feature | 设置/取消推荐 |

### 3. 业务逻辑
- 置顶文章在列表中优先显示
- 推荐文章有特殊标记
- 支持批量操作
