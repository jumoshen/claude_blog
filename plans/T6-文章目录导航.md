# T6: 文章目录/大纲导航

## 需求分析
文章详情页展示文章目录，点击目录项可跳转到对应章节。

## 实现方案

### 1. API 设计
文章详情接口已返回 HTML content，前端可自行解析 headings 生成目录。
如需后端提供目录结构：

| 方法 | 路径 | 说明 |
|-----|------|------|
| GET | /api/v1/posts/:slug/toc | 获取文章目录 |

### 2. 目录结构
```json
{
  "toc": [
    {"level": 2, "text": "简介", "id": "简介"},
    {"level": 2, "text": "安装", "id": "安装"},
    {"level": 3, "text": "Windows", "id": "windows"},
    {"level": 3, "text": "macOS", "id": "macos"}
  ]
}
```

### 3. 实现方式
- 解析 markdown AST 提取 headings
- goldmark 已配置 WithAutoHeadingID 生成 ID
