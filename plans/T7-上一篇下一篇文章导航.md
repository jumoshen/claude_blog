# T7: 上一篇/下一篇文章导航

## 需求分析
文章详情页显示上一篇和下一篇导航，方便读者连续阅读。

## 实现方案

### 1. API 设计
| 方法 | 路径 | 说明 |
|-----|------|------|
| GET | /api/v1/posts/:slug/navigation | 获取文章导航（上一篇、下一篇） |

### 2. 返回结构
```json
{
  "prev": {"slug": "prev-post", "title": "上一篇文章"},
  "next": {"slug": "next-post", "title": "下一篇文章"}
}
```

### 3. 排序规则
- 按发布日期排序，当前文章之前的为下一篇，之后的为上一篇
