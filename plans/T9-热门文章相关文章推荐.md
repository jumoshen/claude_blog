# T9: 热门文章/相关文章推荐

## 需求分析
- 热门文章：按浏览量排序的文章列表
- 相关文章：与当前文章标签/分类相似的文章

## 实现方案

### 1. API 设计
| 方法 | 路径 | 说明 |
|-----|------|------|
| GET | /api/v1/posts/popular | 热门文章列表 |
| GET | /api/v1/posts/:slug/related | 相关文章列表 |

### 2. 热门文章
- 按浏览量降序
- 默认返回10篇

### 3. 相关文章
- 与当前文章共享相同标签
- 按共享标签数量降序
- 默认返回5篇
